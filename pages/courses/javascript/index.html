JavaScript Temelleri

Bu bölümde JavaScript'in temel kavramlarını ve modern özellikleri öğreneceksiniz.

JavaScript'te Değişkenler ve Veri Tipleri

Modern JavaScript'te değişken tanımlama ve temel veri tipleri:

  • let ve const ile değişken tanımlama
  • String, Number, Boolean
  • Array ve Object yapıları
  • Template literals

Modern JavaScript Örneği


// Modern JavaScript özellikleri
const user = {
    name: 'Ahmet',
    age: 25,
    skills: ['JavaScript', 'React', 'Node.js']
};

// Destructuring
const { name, age } = user;

// Template literals
const greeting = `Merhaba, ben ${name}! 
${age} yaşındayım ve aşağıdaki teknolojilerle çalışıyorum:`;

// Array methods
const skillsList = user.skills
    .map(skill => `- ${skill}`)
    .join('\n');

// Modern fonksiyon
const displayProfile = () => {
    console.log(greeting);
    console.log(skillsList);
};

// Promise örneği
const fetchUserData = () => {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            resolve(user);
        }, 1000);
    });
};

// Async/await kullanımı
async function init() {
    try {
        const userData = await fetchUserData();
        displayProfile();
    } catch (error) {
        console.error('Hata:', error);
    }
}

init();
