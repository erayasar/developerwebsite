Python Nedir?

Python, okunabilir ve anlaşılır syntax'ı ile öne çıkan, çok amaçlı bir programlama dilidir. Başlıca özellikleri:

  • Kolay öğrenilebilir syntax
  • Geniş kütüphane desteği
  • Platform bağımsız çalışma
  • Büyük topluluk desteği

# Merhaba Dünya programı
def main():
    """
    Basit bir merhaba dünya örneği
    """
    message = "Merhaba, Python!"
    print(message)
    
    # Kullanıcıdan girdi alma
    name = input("Adınız nedir? ")
    print(f"Merhaba, {name}!")

if __name__ == "__main__":
    main()
