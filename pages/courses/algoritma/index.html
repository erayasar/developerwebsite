Algoritma Nedir?

Bir problemi çözmek için takip edilmesi gereken adımlar bütünüdür. İyi bir algoritma şu özelliklere sahip olmalıdır:

  • Kesinlik
  • Sonluluk
  • Verimlilik
  • Anlaşılırlık

Örnek Algoritma


# İkili arama algoritması
def binary_search(arr: list, target: int) -> int:
    """
    Sıralı bir dizide hedef elemanın indeksini bulur.
    
    Args:
        arr: Sıralı dizi
        target: Aranan eleman
    
    Returns:
        Elemanın indeksi, bulunamazsa -1
    """
    left = 0
    right = len(arr) - 1
    
    while left <= right:
        mid = (left + right) // 2
        
        if arr[mid] == target:
            return mid  # Eleman bulundu
        elif arr[mid] < target:
            left = mid + 1  # Sağ yarıya bak
        else:
            right = mid - 1  # Sol yarıya bak
    
    return -1  # Eleman bulunamadı
