React'e Giriş

Bu bölümde React'in temel kavramlarını ve modern web geliştirmedeki önemini öğreneceksiniz.

React Nedir?

React, Facebook tarafından geliştirilen açık kaynaklı bir JavaScript kütüphanesidir. Başlıca özellikleri:

  • Component tabanlı yapı
  • Virtual DOM
  • JSX syntax
  • Tek yönlü veri akışı
